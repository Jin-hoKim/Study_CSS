1. Parcel Install
    • npm install -g parcel-bundler :: install Parcel via npm
    • npm init -y :: make file "package.json"
  2. Make Index files
    • index.html (required)
    • index.css
    • index.js
  3. Edit file "package.json"
    • "scripts": { "start": "parcel index.html" } :: add source to "package.json"
  4. Run Parcel
    • npm run start :: run parcel server via terminal
